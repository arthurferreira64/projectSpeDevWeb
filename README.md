# projectSpeDevWeb

Le projet est composé d'un frontend est d'un backend

### Lancer le frontend

Se déplacer dans le bon dossier

```powershell, ps1
cd front
```

installer les dépendances

```powershell, ps1
npm install
```

Lancer le frontend

```powershell, ps1
npm run dev
```

### Lancer le backend

Se déplacer dans le bon dossier

```powershell, ps1
cd backend
```

installer les dépendances

```powershell, ps1
npm install
```

Lancer le backend

```powershell, ps1
npm run dev
```

Lancer les test backend

```powershell, ps1
npm run test
```
